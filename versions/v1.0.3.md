## 变更说明

在 post-sync guard 末尾新增 versions/ 检查逻辑：
- 读取本地 _meta.json 中的 version 字段
- 若 versions/v{version}.md 存在但 GitHub 上不存在，则打印 ⚠️ 警告提示
- 此检查为 advisory（不阻断同步），因为 versions/ 快照由 skill-bump 管理
- 目的：让 versions/ 快照缺失从"静默遗漏"变成"可见警告"

## 设计决策

- 为什么 advisory 而非 fail：versions/ 快照由 skill-bump 主管，github-sync-skill 补充提示；强制要求 versions/ 会改变工具边界，增加不必要耦合
