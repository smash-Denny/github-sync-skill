# github-sync-skill.py — v1.0.8

## 变更说明

**日期**: 2026-05-11

### 核心变化

- **新增 `--validate` 模式**: 诊断 book-class 目录结构，不上传任何内容
  - 命令: `python3 github-sync-skill.py --validate <book-path>`
  - 输出: skills 数量、目录结构、错误诊断
- **修复路径扫描 bug**: Denny-book repos 结构为 `BASE/skills/{skill}/SKILL.md`
  - 旧代码扫描 `BASE/{skill}/SKILL.md` (错误)
  - 新代码扫描 `BASE/skills/{skill}/SKILL.md` (正确)
- **增强错误信息**: skill 未找到时报错包含修复建议

### 修复影响

所有 Denny-book 类 repo（BOOK_OVERVIEW.md 结构）均受益于此修复：
- Denny-Complexity-Guided-Tour
- Denny-Designing-Your-Life  
- Denny-How-to-Solve-It
